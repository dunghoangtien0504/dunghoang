# SAB Ads Command

Activate **Facebook Ads Skill** — chuyên gia thực chiến setup campaign, viết ad copy, tối ưu CPC/CPM và scale.

## Agent
Load `agents/02-marketing-agent.md` (chính) + `agents/03-sales-agent.md` (retargeting/conversion)

## Skill
Load `skills/facebook-ads/SKILL.md` và tất cả references.

## Usage
```
/sab-ads [hành động] [context]
```

**Ví dụ:**
- `/sab-ads setup traffic` — Setup campaign Traffic về landing page
- `/sab-ads setup message` — Setup campaign Tin nhắn Messenger
- `/sab-ads setup engagement` — Setup campaign Tương tác/Comment
- `/sab-ads copy [sản phẩm]` — Viết 3-5 ad copy để test
- `/sab-ads review` — Paste số liệu hiện tại → chẩn đoán và tối ưu
- `/sab-ads scale` — Hướng dẫn scale campaign đang win
- `/sab-ads retargeting` — Setup retargeting cho người đã xem trang

## Workflow
1. Xác định user cần gì (setup / copy / optimize / scale)
2. Hỏi thông tin cần thiết (mục tiêu, sản phẩm, số liệu hiện tại)
3. Đưa ra hướng dẫn cụ thể và actionable
4. Với optimize: yêu cầu user paste số liệu thực tế trước khi tư vấn

## Ngân sách mặc định: 150k-250k VND/camp/ngày
## Cấu trúc mặc định: 1 Camp → 1 Ad Group → 3-5 Ads
## Thời gian test: 3-5 ngày trước khi đánh giá
