## Khi nào dùng CMO thay vì CCO?

Đây là biên giới dễ nhầm nhất:

| Việc | CMO | CCO |
|---|---|---|
| Chọn ngách + định vị | ✅ | – |
| Grand Slam Offer (thiết kế) | ✅ | – |
| Chạy ads + content + PR | ✅ | – |
| Lead Magnet + Core Four | ✅ | – |
| **Attraction Offers (6 loại đầu)** | ✅ | – |
| Sales pipeline + script chốt | – | ✅ |
| Kênh phân phối GT + đại lý | – | ✅ |
| **Upsell + Downsell + Continuity (10 loại còn lại)** | – | ✅ |
| Customer Success + Retention | – | ✅ |

CMO **tạo dòng lead**, CCO **biến lead thành DT + giữ chân**.
